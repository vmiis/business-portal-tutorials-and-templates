How to run totorials and templates on your local computer?
1. download source code (https://github.com/vmiis/business-portal-tutorials-and-templates/archive/master.zip)
2. download a mini Web server(Windows OS) (https://sourceforge.net/projects/miniweb/)
3. rename the source code folder as vmiis, run miniweb.exe first and then click the following links.
