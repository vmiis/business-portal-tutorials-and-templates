How to run totorials and templates on your local computer?
1. download source code (https://github.com/vmiis/business-portal-tutorials-and-templates/archive/master.zip)
2. download a mini Web server(Windows OS) (https://cbs.wappsystem.com/dev/download/GitHub.rar)
3. create a folder called vmiis and copy all unziped files to the vmiis folder, run server.bat first and then click the following links.
