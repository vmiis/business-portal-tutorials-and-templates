Host and run applications on your local computer
1. download a mini Web server(Windows OS), unzip it and create a folder structure like following.

2. download an application repository, unzip it, and then copy to a sub directory.
3. run server.bat first and then open a browser and type the application address (for example: http://127.0.0.1:8000/vmiis/bookkeeping/index.html) or click the following links.
